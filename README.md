<hr><h2 align="left">SOBRE</h2>
<p align="left">
    O código lê dois ou mais arquivos PDF, extrai todas as páginas de cada um e grava essas páginas em um novo arquivo PDF único.
</p>
<p align="left">
    Recomendado a instalação:
    
    python.exe -m pip install --upgrade pip

    pip install pypdf
</p>
